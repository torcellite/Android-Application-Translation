Disclaimer

  1. The tool is only an automated version of looking up strings on Google Translate.
  2. It has been modified specifically to translate strings.xml in Android applications.
  3. To exclude the strings or items that you do not want to be translated include the <!-- exclude --> either inbetween the tags.
            For example,
            <string name='app_name'>Name of the application</string>
            will be changed to
            <string name='app_name'><!-- exclude -->Name of the application</string>
            the translator will ignore this string for translation.
  4. Check Store Listing if you'd like to translate text for the Google Play Store. Encode it in a regular xml file and the application will translate it for you without escaping apostrophes.
