Android Application Translate

Tutorial

  1. The tool is only an automated version of looking up strings on Bing Translate.
  2. It has been modified specifically to translate strings.xml in Android applications.
  3. To exclude the strings or items that you do not want to be translated add the "exclude" attribute to the element.
            For example,
            <string name='app_name'>Name of the application</string>
            will be changed to
            <string name='app_name' exclude='contents in here do not matter'>Name of the application</string>
            The translator will ignore this string for translation.
  4. If you have more tags like "list-item" add them in the extra tags field. If you do not have any or are unsure of what this means, please leave the field empty.
  5. Check Store Listing if you'd like to translate text for the Google Play Store. Encode it in a regular xml file and the application will translate it for you without escaping single quotes.
